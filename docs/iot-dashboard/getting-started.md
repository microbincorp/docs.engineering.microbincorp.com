# Getting Started with IoT Dashboard

This guide will help you quickly get up and running with the MicroBin IoT Dashboard.

## Quick Start Guide

1. **Create an Account** - Sign up for a MicroBin IoT Dashboard account through our [registration portal](https://dashboard.microbin.io/signup).

2. **Add Devices** - Connect your first IoT device using the device registration wizard:
   - Navigate to the "Devices" tab in your dashboard
   - Click "Add New Device"
   - Follow the guided setup process
   - Generate and save your device credentials

3. **Create a Dashboard** - Use the dashboard builder to create your first visualization:
   - Click "New Dashboard" from the main menu
   - Choose a layout template or start from scratch
   - Name your dashboard and set access permissions

4. **Customize** - Add widgets, set up alerts, and customize your dashboard layout:
   - Drag and drop widgets from the widget library
   - Connect widgets to your device data sources
   - Configure widget appearance and behavior
   - Set up alerts and notifications for important events
