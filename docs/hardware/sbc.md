# Single Board Computers

MicroBin's Single Board Computers (SBCs) provide complete computing solutions on a single circuit board. These powerful and compact computers are ideal for a wide range of applications including embedded systems, IoT gateways, edge computing, and more.

## Available Models

| Model | Processor | RAM | Storage | Key Features |
|-------|-----------|-----|---------|-------------|
| MB-SBC100 | Quad-core ARM Cortex-A53 | 2GB | 16GB eMMC | Wi-Fi, Bluetooth, Gigabit Ethernet |
| MB-SBC200 | Octa-core ARM Cortex-A72/A53 | 4GB | 32GB eMMC | Wi-Fi 6, Bluetooth 5.2, Dual Gigabit Ethernet |
| MB-SBC300 | Hexa-core ARM Cortex-A78/A55 | 8GB | 64GB eMMC | Wi-Fi 6E, Bluetooth 5.3, 2.5GbE, PoE |

## Technical Specifications

### Power Requirements
- Input voltage: 5-12V DC
- Power consumption: 2-15W depending on load and model

### Connectivity
- USB 3.0/2.0 ports
- HDMI output
- Ethernet
- Wireless networking (model dependent)
- GPIO expansion headers

## Getting Started

1. **Power Supply Connection**
   Connect the appropriate power supply to the designated power input connector.

2. **Peripheral Connection**
   Connect required peripherals (keyboard, mouse, display) via the available ports.

3. **Operating System Installation**
   Follow the [OS installation guide](/hardware/os-installation) to set up your preferred operating system.

4. **GPIO Configuration**
   Configure GPIO pins according to your project requirements using our [GPIO library](/hardware/gpio-library).
